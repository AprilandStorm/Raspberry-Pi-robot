sudo apt update
sudo apt install -y libpoco-dev libcamera-apps i2c-tools
✅ 可以成功在树莓派（Raspberry Pi OS / Debian）上安装 Poco 库的开发文件。

✅ 详细说明：
1. libpoco-dev
这是 POCO C++ Libraries 的开发包。
安装后会包含：
所有 POCO 头文件（如 <Poco/Net/HTTPClientSession.h>）
静态库和链接所需的符号
用于编译和链接 C++ 程序
📌 你写代码时 #include <Poco/...> 就依赖这个包。

2. 安装的 POCO 模块（自动包含）
libpoco-dev 是一个元包，它会自动安装以下核心模块：

libpoco-foundation-dev
libpoco-net-dev
libpoco-crypto-dev
libpoco-util-dev
libpoco-xml-dev
libpoco-zip-dev
这些提供了：

网络（HTTP、TCP、UDP）
文件配置（XML、JSON）
多线程
加密
日志
进程间通信等
3. 其他你安装的包
包名	用途
libcamera-apps	安装 libcamera 的命令行工具（如 libcamera-hello, libcamera-still），用于摄像头控制
i2c-tools	安装 i2cdetect, i2cget, i2cset 等工具，用于调试 I2C 设备
⚠️ 注意：libcamera-apps 和 POCO 无关，但对嵌入式开发很有用。

✅ 编写一个简单的 POCO 测试程序
cpp
深色版本
// test_poco.cpp
#include <Poco/DateTime.h>
#include <Poco/Timestamp.h>
#include <iostream>

int main() {
    Poco::Timestamp now;
    Poco::DateTime dt(now);
    std::cout << "当前时间: " 
              << dt.year() << "-" 
              << dt.month() << "-" 
              << dt.day() << " "
              << dt.hour() << ":"
              << dt.minute() << ":"
              << dt.second()
              << std::endl;
    return 0;
}
编译（必须链接 POCO 库）
bash
深色版本
g++ -o test_poco test_poco.cpp -lPocoFoundation
📌 使用 -lPocoFoundation 链接核心库，其他功能可能需要额外链接，如 -lPocoNet。

✅ 常见用途（POCO 在树莓派上的典型应用）
用途	所需库
HTTP 客户端/服务器	-lPocoNet
JSON 配置读写	-lPocoUtil -lPocoJSON
多线程任务	-lPocoFoundation
日志记录	-lPocoFoundation
TCP/UDP 通信	-lPocoNet
